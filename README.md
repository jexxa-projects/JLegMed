# JLegMed
Prototype for developing a new library 

## Requirements

*   Java 17+ installed
*   IDE with maven support 
 

## Build the Project

*   Checkout the new project in your favorite IDE

*   Without running integration tests:
    ```shell
    mvn clean install 

    java -jar target/jlegmed-jar-with-dependencies.jar
    ```
